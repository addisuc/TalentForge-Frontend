## Job Management

### Create New Job

1. Click **"Create Job"**
2. Enter job details:
   - **Job Title**: Position name
   - **Department**: Team/department
   - **Location**: City, state, or remote
   - **Job Type**: Full-time, Part-time, Contract
   - **Experience Level**: Entry, Mid, Senior, Executive
   - **Salary Range**: Min/max compensation
3. Add job description:
   - Overview
   - Responsibilities
   - Requirements
   - Qualifications
   - Benefits
4. Set application settings:
   - Application deadline
   - Screening questions
   - Required documents
5. Click **"Publish Job"**

### Edit Job

1. Go to **"Jobs"**
2. Click job to edit
3. Click **"Edit"**
4. Update information
5. Click **"Save Changes"**

### Close Job

1. Open job details
2. Click **"Close Job"**
3. Select reason:
   - Position filled
   - No longer needed
   - On hold
4. Confirm

### Duplicate Job

1. Open job details
2. Click **"Duplicate"**
3. Modify details
4. Publish

## Application Management

### Review Applications

1. Go to **"Applications"**
2. Filter by:
   - Job
   - Status
   - Date range
   - Rating
3. Click application to review

### Application Review

1. View candidate profile:
   - Resume
   - Cover letter
   - Work experience
   - Education
   - Skills
2. Review screening answers
3. Check application timeline

### Move Application Through Pipeline

**14-Stage Pipeline:**

1. **Applied** - Initial submission
2. **Screening** - Initial review
3. **Phone Screen** - First contact
4. **Assessment** - Skills test
5. **First Interview** - Initial interview
6. **Second Interview** - Follow-up
7. **Final Interview** - Last round
8. **Reference Check** - Verify references
9. **Background Check** - Verification
10. **Offer Preparation** - Preparing offer
11. **Offer Extended** - Offer sent
12. **Offer Accepted** - Candidate accepted
13. **Hired** - Onboarding
14. **Rejected** - Not selected

**To Move Application:**
1. Open application
2. Click **"Change Status"**
3. Select new status
4. Add notes (optional)
5. Click **"Update"**

### Rate Candidates

1. Open application
2. Click **"Rate Candidate"**
3. Select rating (1-5 stars)
4. Add comments
5. Click **"Save"**

### Add Notes

1. Open application
2. Click **"Add Note"**
3. Enter note text
4. Mark as private (team only) or shared
5. Click **"Save"**

### Reject Application

1. Open application
2. Click **"Reject"**
3. Select rejection reason
4. Choose to send notification
5. Confirm

## Interview Scheduling

### Schedule Interview

1. Open application
2. Click **"Schedule Interview"**
3. Enter details:
   - **Interview Type**: Phone, Video, In-person
   - **Date and Time**
   - **Duration**: 30, 45, 60 minutes
   - **Interviewer(s)**: Select team members
   - **Location/Meeting Link**
   - **Instructions**: Notes for candidate
4. Click **"Schedule"**
5. Notifications sent automatically

### Reschedule Interview

1. Open interview
2. Click **"Reschedule"**
3. Select new date/time
4. Add reason
5. Click **"Update"**
6. Candidate notified automatically

### Cancel Interview

1. Open interview
2. Click **"Cancel"**
3. Provide reason
4. Confirm
5. Candidate notified

### Join Video Interview

1. Go to **"Interviews"**
2. Click **"Join"** (5 min before)
3. Test audio/video
4. Start interview

### Add Interview Feedback

1. After interview, click **"Add Feedback"**
2. Rate candidate
3. Enter detailed feedback
4. Recommend next steps
5. Click **"Submit"**

## Client Management

### Add New Client

1. Go to **"Clients"**
2. Click **"Add Client"**
3. Enter:
   - Company name
   - Contact person
   - Email
   - Phone
   - Industry
   - Company size
4. Click **"Save"**

### Manage Client Relationships

1. Open client profile
2. View:
   - Contact information
   - Active jobs
   - Placement history
   - Communication log
3. Add notes and activities

### Client Job Requests

1. Receive job request from client
2. Review requirements
3. Create job posting
4. Link to client
5. Update client on progress

### Submit Candidates to Client

1. Open candidate profile
2. Click **"Submit to Client"**
3. Select client and job
4. Add cover note
5. Click **"Submit"**
6. Client receives notification

## Reports & Analytics

### Dashboard Metrics
