# TalentForge User Manuals

This directory contains user manuals for different user roles in TalentForge.

## Available Manuals

- **[Candidate User Manual](./candidate-user-manual.md)** - For job seekers
- **[Recruiter User Manual](./recruiter-user-manual.md)** - For recruiters and tenant admins
- **[Client User Manual](./client-user-manual.md)** - For client portal users
- **[Platform Admin User Manual](./platform-admin-user-manual.md)** - For platform administrators
